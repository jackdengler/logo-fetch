Ticker → Logo Fetch

Paste tickers, get transparent PNGs of each company's icon with its name beside it. The text colour is sampled from the icon.

Wordmark providers (optional — improves logo quality)

By default this site uses Clearbit + DuckDuckGo + Google as icon sources. Adding a token from Brandfetch or logo.dev gives you sharper, higher-res icons (Google favicons are small). Tokens are stored in your browser only.

Icon source fallback: Brandfetch → logo.dev → Clearbit → DuckDuckGo → Google. The composited PNG is always 1024×256 with a transparent background; the company name is drawn beside the icon in a colour sampled from it.
