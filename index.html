Wordmark providers (optional API keys)

Adding a token from Brandfetch or logo.dev gives sharper icons. Tokens are stored in your browser only (localStorage).

Icon source fallback: SimpleIcons → Brandfetch → logo.dev → Clearbit → DuckDuckGo → Google (first to return a usable image wins). Each PNG has a transparent background and auto-fits the icon plus the company name, drawn in a colour sampled from the icon. Font size is in points (like PowerPoint).
